# Ninj-OS Vanilla Bridge

This behavior pack lets a Mojang Bedrock Dedicated Server participate in Ninj-OS Full Proxy Mode without Endstone. It consumes a one-use identity grant, restores the current player's command permission level, assigns verified role tags, and rejects direct joins when strict mode is enabled.

See `docs/VANILLA_BRIDGE_INSTALL.md` in the main repository for the complete installation and security guide.

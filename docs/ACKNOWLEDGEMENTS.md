# Acknowledgements

Ninj-OS Proxie Edge Fabric was developed with reference to and inspired by
**ProxyPass by SculkCatalystMC**.

- Project: ProxyPass
- Maintainer: SculkCatalystMC
- Repository: https://github.com/SculkCatalystMC/ProxyPass
- Upstream license: GNU Affero General Public License v3.0

ProxyPass provided an important reference point for Minecraft Bedrock proxying,
identity relay, packet forwarding, and the practical requirements of placing a
proxy between Bedrock clients and backend servers.

Ninj-OS Proxie is a separately maintained implementation with its own edge engine,
routing model, Pterodactyl deployment flow, dashboard control plane, SQLite data
model, transfer-ticket system, security controls, and Endstone companion tooling.
It is not an official ProxyPass release and is not represented as endorsed by
SculkCatalystMC.
